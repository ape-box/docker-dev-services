# Docker Dev Services

## Description

This is a set of scripts to run services used in development with
Docker Compose.

Supported services:

- ELK stack (Elasticsearch + Logstash + Kibana)
- MongoDB
- RabbitMQ
- Redis
- EventStore

## Prerequisites

- Docker with Docker Compose

## Usage

Start all the services:

```
start.cmd
```

Stop all the services and remove containers:

```
stop.cmd
```
